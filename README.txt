This program uses OOP.

Class Card will create a card after shuffling and moving cards

Class PokerHand forms a poker hand from a user-determined number of decks and cards per hand.
	each function in PokerHand is used to identify the hands dealt by the program
	classify() will then identify the best poker hand in the hand dealt
	a table of each hand dealt and hand returned will be printed to the terminal, calculations are equivalent to https://en.wikipedia.org/wiki/Poker_probability
